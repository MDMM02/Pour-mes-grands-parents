Scanner un QR code

Billets, menus, Wi‑Fi : scanner sans risque.

Objectif Scanner un QR code (billets, menu, Wi‑Fi, lien) en toute sécurité.

Le plus simple

Souvent, il suffit d’ouvrir Appareil photo et de viser le QR code.

  1. Ouvre Appareil photo.
  2. Vise le QR code (sans bouger).
  3. Un lien apparaît → appuie dessus.

Si ça ne marche pas

  • Plus de lumière, ou rapproche-toi un peu.
  • Nettoie l’objectif (tissu doux).
  • Sur certains téléphones : option “Scanner QR”.

QR code Wi‑Fi

  1. Scanne le QR code.
  2. Appuie sur Rejoindre le Wi‑Fi (si proposé).
  3. Vérifie le nom du réseau avant d’accepter.

Billets / menus

  1. Scanne avec l’appareil photo.
  2. Appuie sur le lien.
  3. Lis le nom du site : s’il est bizarre, n’insiste pas.

Sécurité (très important)

Un QR code peut mener à un faux site. Un QR code collé par-dessus un autre = méfiance.

  • Regarde l’adresse avant de valider (site officiel si possible).
  • Ne donne jamais : mot de passe, code SMS, infos bancaires.
  • Si on te demande de payer alors que ce n’était pas prévu : stop et demande de l’aide.

Mini-exercices

  1. Scanne un QR code “officiel”.
  2. Vérifie que tu vois un lien avant de cliquer.
  3. Si c’est louche : ferme et demande de l’aide.
Astuce : sur une version imprimée, surligne les boutons importants (caméra, micro, envoyer).
