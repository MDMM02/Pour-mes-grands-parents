Agrandir le texte / Loupe

Rendre le téléphone plus lisible et plus simple.

Objectif Agrandir le texte, activer la loupe et quelques options d’accessibilité.

iPhone : agrandir le texte

  1. Réglages → Affichage et luminosité.
  2. Taille du texte → glisse le curseur.
  3. Option : Texte en gras.

Android : agrandir le texte

  1. Paramètres → Affichage.
  2. Taille de police (ou “Taille du texte”).
  3. Option : Taille d’affichage (icônes plus grandes).

Loupe / zoom (très utile)

iPhone

  1. Réglages → Accessibilité.
  2. Loupe → activer.
  3. Ensuite : triple-clic bouton latéral pour l’ouvrir (selon réglage).

Android

  1. Paramètres → Accessibilité.
  2. Agrandissement / Loupe.
  3. Activer le raccourci (bouton accessibilité).

Autres options pratiques

  • Contraste / “Amélioration visibilité”.
  • Lecture à voix haute (selon téléphone).
  • Son : augmenter le volume, activer “Aides auditives” si besoin.

Mini-exercices

  1. Agrandis le texte d’un cran.
  2. Active la loupe, puis teste-la sur une petite écriture.
  3. Remets un réglage si c’est trop grand.
Astuce : fais une première fois avec un proche, puis refais seul(e) le lendemain.
